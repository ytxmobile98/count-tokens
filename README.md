# Count Tokens for Files

A utility tool for counting tokens in files, written in TypeScript. It uses the `tiktoken` library to count tokens based on the specified model (default is "`gpt-4`"). The tool can be run from the command line, and it will display the token count for each file provided as an argument.

## Usage

### Building the Project

```bash
pnpm install
pnpm build
```

### Running the Tool

```bash
pnpm start FILE_PATH [FILE_PATH...]
```
